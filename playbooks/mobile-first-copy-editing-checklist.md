# Mobile-First Copy Editing Checklist

Use this before publishing any landing page or article.

## 1) First-screen clarity (5-second test)
- [ ] Who is this for?
- [ ] What value is offered?
- [ ] What should I do next?

## 2) Sentence weight
- [ ] No sentence over ~20 words unless needed
- [ ] One idea per sentence
- [ ] Remove filler words that add no meaning

## 3) Paragraph rhythm
- [ ] Prefer 2–4 lines per paragraph on mobile
- [ ] Split dense blocks into bullets where useful
- [ ] Keep transitions explicit

## 4) CTA clarity
- [ ] Primary CTA visible without hunting
- [ ] CTA text action-oriented and human
- [ ] No competing primary CTAs in same section

## 5) Trust signals
- [ ] Contact path obvious
- [ ] Legal links reachable
- [ ] Claims are concrete (avoid vague superlatives)

## 6) Real-device pass
- [ ] iPhone-sized viewport checked
- [ ] Android-sized viewport checked
- [ ] No awkward wraps in labels/meta rows

## 7) Final trim
- [ ] Remove one non-essential sentence per major section
- [ ] Re-read hero + CTA only: does flow still work?

## Example quick edits
- Before: "In this transformative journey we lovingly guide you toward..."
- After: "We guide you with clear steps and personal support."

- Before: "Please feel free to potentially contact us if you might want..."
- After: "Kontaktiere mich gerne, wenn du starten möchtest."
